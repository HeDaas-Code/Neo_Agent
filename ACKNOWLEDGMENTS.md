# 致谢 / Acknowledgments

本项目使用或参考了以下开源项目，我们在此表示衷心的感谢：

## 核心依赖 / Core Dependencies

### MemU
- **项目地址**: https://github.com/NevaMind-AI/memU
- **许可证**: Apache License 2.0
- **用途**: 24/7主动智能体的记忆管理框架
- **说明**: MemU是一个为长期运行智能体设计的记忆框架，能够持续捕获和理解用户意图，大大降低了长期运行的LLM token成本。我们使用MemU替代了原有的基于LLM的自动记忆总结功能，提供了更高效的记忆管理机制。

### LangChain
- **项目地址**: https://github.com/langchain-ai/langchain
- **许可证**: MIT License
- **用途**: 构建LLM应用的核心框架

### LangGraph
- **项目地址**: https://github.com/langchain-ai/langgraph
- **许可证**: MIT License
- **用途**: 多智能体协作的状态图管理

### DeepAgents
- **项目地址**: https://github.com/aiwaves-cn/agents
- **许可证**: Apache License 2.0
- **用途**: 子智能体生成、长期记忆和文件系统功能增强
