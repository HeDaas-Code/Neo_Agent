# Neo Agent Qt GUI 使用指南

## 简介

Neo Agent Qt GUI 是一个仿QQ风格的现代化聊天界面，用于与智能代理进行对话。

## 功能特性

### 主要功能

- ✅ **QQ风格界面** - 仿照QQ聊天软件设计的现代化界面
- ✅ **消息气泡** - 用户消息（绿色）和AI消息（白色）气泡显示
- ✅ **时间戳** - 每条消息显示发送时间
- ✅ **表情支持** - 内置表情选择器，支持常用emoji
- ✅ **侧边栏** - 显示AI角色信息、头像、在线状态
- ✅ **Debug面板** - 可通过菜单启用/禁用调试信息面板
- ✅ **响应式布局** - 自适应窗口大小，处理超长文本
- ✅ **异步处理** - 使用多线程处理AI响应，UI不会卡顿
- ✅ **快捷键** - Ctrl+Enter发送消息

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境

```bash
cp example.env .env
# 编辑 .env 配置API密钥和角色信息
```

### 3. 运行GUI

```bash
python run_qt.py
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 发送消息 |
| `Ctrl+Q` | 退出应用 |

## 对比原有Tkinter GUI

### Qt GUI优势

1. **更现代的外观** - 仿QQ的专业级UI设计
2. **更好的性能** - Qt的渲染性能优于Tkinter
3. **更流畅的体验** - 异步处理，无卡顿
4. **更丰富的功能** - 表情、时间戳、状态指示等
5. **更好的可扩展性** - Qt提供更多组件和功能
