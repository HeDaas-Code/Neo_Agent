# Live2D助手快速开始指南

## 5分钟快速上手 🚀

### 第一步：安装 (1分钟)

```bash
# 克隆仓库（如果还没有）
git clone https://github.com/HeDaas-Code/Neo_Agent.git
cd Neo_Agent

# 切换到Live2D助手分支
git checkout copilot/add-live2d-assistant-character

# 安装依赖
pip install -r requirements.txt
```

### 第二步：配置 (2分钟)

```bash
# 复制配置文件
cp example.env .env

# 编辑.env文件（可选）
# 如果需要使用智能对话功能，请配置API密钥
# nano .env  或  notepad .env
```

最小配置（可以不改，使用默认值）：
```env
CHARACTER_NAME=小可
DEBUG_MODE=True
```

### 第三步：启动 (30秒)

**Linux/Mac:**
```bash
./start_live2d.sh
```

**Windows:**
```batch
start_live2d.bat
```

**或直接运行:**
```bash
python live2d_assistant.py
```

### 第四步：开始使用 (1分钟)

1. **与小可打招呼** 😊
   - 点击"💬 聊天"标签
   - 输入"你好，小可！"
   - 按Enter发送

2. **开始第一个番茄时钟** 🍅
   - 点击"🍅 番茄时钟"标签
   - 点击"开始工作"按钮
   - 专注25分钟！

3. **创建第一个日程** 📅
   - 点击"📅 日程"标签
   - 点击"+ 新建日程"
   - 填写标题和描述
   - 保存

4. **记录第一条笔记** 📝
   - 点击"📝 笔记"标签
   - 点击"+ 新建笔记"
   - 写下你的想法
   - 保存

5. **创建第一个计划** 🎯
   - 点击"🎯 计划"标签
   - 点击"+ 新建计划"
   - 设定你的目标
   - 保存

## 故障排查 🔧

### 问题1: 无法启动应用

**症状**: 双击start_live2d.sh无反应

**解决方案**:
```bash
# 添加执行权限
chmod +x start_live2d.sh

# 或直接运行Python
python live2d_assistant.py
```

### 问题2: 提示缺少依赖

**症状**: ImportError: No module named 'xxx'

**解决方案**:
```bash
# 重新安装依赖
pip install -r requirements.txt
```

### 问题3: 智能对话不工作

**症状**: 小可不回复消息

**解决方案**:
1. 检查.env文件中的API配置
2. 确保SILICONFLOW_API_KEY已正确设置
3. 如果不需要对话功能，可以不配置API密钥

### 问题4: 窗口太小/太大

**解决方案**:
在live2d_assistant.py中修改：
```python
self.geometry("400x650")  # 修改为你想要的尺寸
```

### 问题5: 数据丢失

**解决方案**:
```bash
# 数据保存在chat_agent.db中
# 定期备份此文件
cp chat_agent.db chat_agent_backup.db
```
