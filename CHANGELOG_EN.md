## [5.0.0] - 2025-01-15

### Added ⭐
- **Emotional Relationship Analysis System**
  - Automatic emotional analysis (triggered every 10 rounds of conversation)
  - 5-dimensional emotional assessment (intimacy, trust, pleasure, resonance, dependence)
  - Automatic relationship type recognition (acquaintance/friend/soulmate, etc.)
  - Emotional tone analysis (positive/neutral/negative)
- **Emotional Visualization**
  - Pentagon emotional radar chart
  - Tab switching with topic timeline
  - Real-time emotional state update
- **AI Tone Adaptation**
  - Automatically adjust conversation tone based on emotional state
  - Generate personalized tone prompts
  - Emotional data persistent storage
- **Debug Support**
  - Complete emotional analysis process logging
  - Detailed tone adjustment tracking

### Optimized
- Optimized visualization area layout
- Improved GUI tab switching experience
- Enhanced emotional analysis accuracy

### Documentation
- Added detailed explanation of emotional analysis system
- Added emotional radar chart usage guide
- Updated FAQ

## Version Explanation

### Version Number Format: Major.Minor.Patch

- **Major**: Incompatible API changes
- **Minor**: Backward-compatible feature additions
- **Patch**: Backward-compatible bug fixes

### Change Types

- **Added**: New features
- **Fixed**: Bug fixes
- **Optimized**: Performance or experience improvements
- **Documentation**: Documentation updates
- **Deprecated**: Features to be removed soon
- **Removed**: Removed features
- **Security**: Security-related fixes
